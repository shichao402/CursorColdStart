# CursorColdStart

> ⚠️ **此项目已归档** - 功能已迁移到 [Dec](https://github.com/shichao402/Dec)

## 迁移说明

CursorColdStart 的规则管理功能已完全合并到 Dec 项目中。请使用 Dec 获取更好的体验：

### 安装 Dec

```bash
# macOS/Linux
curl -fsSL https://raw.githubusercontent.com/shichao402/Dec/main/scripts/install.sh | bash

# 或从 Release 下载
# https://github.com/shichao402/Dec/releases
```

### 功能对照

| CursorColdStart | Dec | 说明 |
|-----------------|-----|------|
| `coldstart init` | `dec init` | 初始化项目配置 |
| `coldstart list` | `dec list` | 列出可用包 |
| 规则生成 | `dec sync` | 同步规则和 MCP 配置 |

### Dec 新特性

- **MCP 支持** - 集成 Model Context Protocol，AI 可直接调用工具
- **多 IDE 支持** - 支持 Cursor 和 CodeBuddy
- **包管理** - 支持规则包和 MCP 工具包
- **本地开发** - `dec link` 支持本地包开发
